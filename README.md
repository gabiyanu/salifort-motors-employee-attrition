# Salifort Motors — Employee Attrition Prediction

**Google Advanced Data Analytics Capstone | Course 7**

## Overview
Built a machine learning model to predict employee attrition for Salifort Motors HR department, 
using a dataset of 15,000 employees across 10 variables including satisfaction level, 
performance review scores, workload, tenure, and salary band.

## Key Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Feature engineering & data cleaning
- Logistic Regression / Random Forest / XGBoost (classification)
- Model evaluation: precision, recall, F1, AUC-ROC
- Business recommendations from model insights

## Dataset
[Kaggle — HR Analytics and Job Prediction](https://www.kaggle.com/datasets/mfaisalqureshi/hr-analytics-and-job-prediction)

## Tools
Python · Pandas · Scikit-learn · Matplotlib · Seaborn
